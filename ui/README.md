# Clinical Decision Support Dashboard
 
A React-based interface for the AI-CDSS, featuring patient data management and evidence-based chat.
 
## Features
- **Patient Context Sidebar**: Upload and manage patient PDF reports.
- **Clinical Chat**: Markdown-rendered chat with citation support.
- **Reference Viewer**: View sources for AI generation.
 
## Setup
 
```bash
cd ui
npm install
npm run dev
```
 
- Open http://localhost:5173
- Ensure backend is running on port 8000.
